## Features

- [x] Read-only access to user collections and games
- [x] TODO! Searching
- [x] TODO! Forums, Videos, Users, etc...

## Requirements

- iOS 9.0+
- Xcode 7.3+
- Swift 2.2

## Dependencies

- [Alamofire 3.3+](https://github.com/Alamofire/Alamofire)
